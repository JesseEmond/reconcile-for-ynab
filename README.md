# ynab-reconcile

## Project setup
```
yarn install
```

Create a `.env.local.` file with the following contents:
```
VUE_APP_YNAB_OAUTH_CLIENT_ID = '...'
```
Where `...` is your client ID, in your YNAB developer settings.

yarn serve
